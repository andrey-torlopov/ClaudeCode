# Architecture

**Applies to:** Весь Rust-код проекта

## Правила

- Не предлагай архитектурные паттерны без запроса
- При рефакторинге сохраняй существующую архитектуру проекта
- Каждый подпроект (lab) - отдельный cargo crate

## What to look for in review

- Предложения сменить архитектуру без явного запроса от пользователя
- Рефакторинг, ломающий существующую архитектурную структуру
- Смешивание нескольких подпроектов в одном crate
