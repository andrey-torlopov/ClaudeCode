# SKILL.md - Шаблон инструментального навыка

> **Назначение:** Инструкция для типовой задачи. Нужно сделать ревью инфраструктуры - достает инструкцию по infra review. Нужно проанализировать зависимости - берет другой инструмент.

## Лимит размера: <=500 строк

**SKILL.md не должен превышать 500 строк.** Если больше - разбивай на:
- `references/*.md` - примеры, таблицы, чек-листы
- `scripts/*.py` - исполняемый код

---

## Структура директории skill

```
.ai/skills/{skill-name}/
├── SKILL.md              # Уровни 1-2: заголовок + инструкция (<=500 строк)
├── scripts/              # Уровень 3: исполняемый код (опционально)
│   ├── generate.py
│   └── validate.sh
└── references/           # Уровень 3: документация (опционально)
    ├── checklist.md
    └── examples.json
```

### Реальный пример: infra-review

```
.ai/skills/infra-review/
├── SKILL.md                    # ~250 строк - core логика
└── references/
    ├── infra-checklist.md      # Полный чек-лист проверок
    └── security-rules.md       # Правила безопасности инфраструктуры
```

## Примеры description

**Хорошо:**
```yaml
description: Ревьюит инфраструктурный код на безопасность, best practices и архитектурные проблемы
description: Сканирует DevOps-репозиторий и каталогизирует структуру, зависимости, стек
description: Проверяет Docker-образы и Terraform-провайдеры на актуальность и конфликты
```

**Плохо:**
```yaml
description: Этот навык предназначен для...  # слишком длинно
description: Помогает с инфраструктурой      # слишком абстрактно
description: Terraform проверка              # нет глагола
```
